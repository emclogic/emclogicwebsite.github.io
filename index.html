Sobre a Emc Logic

Fundada em 2017, a Emc Logic é uma empresa que presta serviços de consultoria, treinamento e desenvolvimento de software para Sistemas Embarcados microcontrolados e microprocessados. Viabilizamos um rápido time to market para dispositivos embarcados incorporando as melhores práticas de desenvolvimento e migrando projetos para os mais atuais softwares open source. Atuamos com projetos de diversas áreas, de equipamentos médicos até tecnologias agrícolas, e estamos sempre dispostos a assessorar nossos clientes para que encontrem soluções rápidas, otimizadas, seguras e com um bom custo benefício.

Serviços

Desenvolvimento de Software para Linux Embarcado

Desenvolvimento de Software em Linux Embarcado para dipositivos que demandam interfaces ricas, assim como alta conectividade e necessidade de integração com diversos periféricos.

  • Programação em Assembly, C/C++, Python e Shell script.
  • Personalização e otimização de imagens com Yocto Project e Buildroot.
  • Desvolvimento de interfaces gráficas para displays TFT ou LVDS.
  • Otimização do tempo de Boot.
  • Desenvolvimento de BSPs.
  • Desenvolvimento de drivers de dispositivos Linux
  • Modificação e configuração de Device-Tree para processadores ARM e RISC-V.
  • Preparação do ambiente de programação e compilação cruzada.

Desenvolvimento de Firmware para Microcontroladores

Desenvolvimento de firmware para microcontroladores de 8 a 64 bits (STM32, Microchip, Texas, NXP, Renesas e etc) garantindo o melhor desempenho para o seu produto.

  • Firmware para sistemas em Baremetal ou RTOS.
  • Redução de footprint em memória e otimização do consumo de energia.
  • Desenvolvimento de drivers e bibliotecas de dispositivos.
  • Configuração de periféricos como UART, I2C, PWM, GPIO, DMA e CAN.
  • Arquitetura modular e desenvolvimento com camadas de abstração - HAL
  • Analisadores estáticos de código e testes unitários.
  • Versionamento de código e deploy seguro.

Consultoria e Treinamentos sob Demanda

Assessoramento para o seu projeto a fim de encontrar estratégias de desenvolvolvimento que estejam alinhandas com as necessidades do mercado. Treinamentos sob demanda da equipe e gestão de conhecimento.

  • Análise da melhor arquitetura de hardware para seu projeto.
  • Escolha das melhores ferramentas e tecnologias para seu projeto.
  • Adequação da melhor estratégia para desenvolvimento de software embarcado rápido e seguro.
  • Documentação e Gestão de conhecimento
  • Assessoria referente à licenças de software livre
  • Treinamentos online e presenciais sob demanda da equipe do cliente.
  • Internalização gradual do desenvolvimento pela equipe do cliente.
