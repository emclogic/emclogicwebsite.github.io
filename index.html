Sobre a Emc Logic

Fundada em 2017, a Emc Logic é uma empresa de consultoria focada no desenvolvimento, viabilização e manutenção de Sistemas Computacionais. Nossa área de atuação vai desde Sistemas Embarcados de baixo nível até ambientes remotos e distribuídos em nuvem. Trabalhamos com diversas ferramentas de software e possuímos especialidade em linguagens e e frameworks como C/C++, Python, Node.js, Shellscript, Yocto e QT e buscamos incorporar as mais atuais ferramentas Open Source de mercado, com foco especial para sistemas que suportem Linux. Conheça mais sobre nós e como podemos viabilizar seus projetos novos ou já existentes.

Serviços

Desenvolvimento de Software para Linux Embarcado

Desenvolvimento de Software em Linux Embarcado para dipositivos que demandam interfaces ricas, alta conectividade e necessidade de integração com diversos periféricos.

  • Programação em C, C++, Python e Shell Script.
  • Personalização e otimização de images com Yocto Project.
  • Desvolvimento de interfaces gráficas para displays TFT ou LVDS.
  • Otimização do tempo de Boot.
  • Desenvolvimento de BSPs.
  • Modificação e configuração de Device-Tree para processadores ARM.
  • Preparação do ambiente de programação e compilação cruzada.

Desenvolvimento de Firmware para Microcontroladores

Desenvolvimento de firmware para microcontroladores de 8 a 64 bits (STM32, Microchip, Texas, NXP, Renesas e etc) garantindo o melhor desempenho para o seu produto.

  • Software para sistemas em Baremetal ou RTOS.
  • Redução de footprint em memória e otimização do consumo de energia.
  • Configuração de periféricos como UART, I2C, PWM, GPIO, DMA e CAN além de desenvolvimento de drivers
  • Arquitetura modular e desenvolivmento com camadas de abstraçao - HAL
  • Analisadores estáticos de código, testes unitários e versionamento de código
